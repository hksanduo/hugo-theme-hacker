# Hugo theme hacker
Hacker 简约风格主题，主要参考[https://github.com/VVelox/hugo-dusky-neon-potato]，个人博客使用，各位如果感兴趣，欢迎pr。

## Overview

* Support for tags


## Usage

Use hugo's -t hugo-theme-hacker or --theme=hugo-theme-hacker option with hugo commands. Example:

```shell
$ hugo server -t hugo-theme-hacker -w -D
```
